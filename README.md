# 🤖 Office Management Telegram Bot

This is a Telegram bot for office attendance and time tracking system.

## 📌 Features
- 🟢 Start Work
- ☕ Break
- 🪑 Back to Seat
- 🔴 Off Work
- ⏱ Time tracking for each action
- 👤 User name and ID tracking
- 📊 Work report system

---

## ⚙️ Tech Stack
- Python
- python-telegram-bot
